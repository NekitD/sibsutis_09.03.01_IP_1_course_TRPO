Игра "Жизнь".

Об игре:
Игра "Жизнь" - клеточный автомат, придуманный английским математиком Джоном Конвеем в 1970 году. Это игра без игроков, в которой человек создаёт начальное состояние, а потом лишь наблюдает за её развитием. Данный проект является одной из его компьютерных реализаций.

Системные требования:
Операционная система Windows
Установленный компилятор Dev C++ TDM-GCC 4.9.2 (64-bit)
Подключенная к нему библиотека SFML 2.4.2 (GCC 4.9.2 TDM (SJLJ) - 64-bit)

Склонируйте репозиторий с Git: git clone https://git.csc.sibsutis.ru/trpo2024/rgr-grjazin-ip312s06
